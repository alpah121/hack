Design Page for hacker simulation window

This website will provide a boilerplate for other websites showcasing security flaws.

Features

  • admin - a place for the admin to view and control: members, tutorials, challenges
  • hack it up - a seperate window where guests can try out pieces of code on pretend victim websites
  • tutorials - a simple webpage with example code. shows: exploits, stats, and the hack it up emulator
  • members - shows percentage completed, show completed challenges, show other flaws might be interested in

Pages

  • /admin/dashboard - view new members, shortcut for new tutorial, unique website visitors
  • /hiu/:id - hack it up emulator
  • /tutorial/:title - a tutorial page
  • /user/dashboard - view challenges, exploits might be interested in, percentage completed
  • /user/hiu - shows last code the user was working on
